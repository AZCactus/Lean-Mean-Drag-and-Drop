About

Lean & Mean Drag and Drop is a small script making drag and drop for re-ordering complex html layout a breeze

LMDD is light weighted (~3kb gzipped) and has zero dependencies. It can work with any type of layout, including flex-box, and produce beautiful looking UI with smooth transitions on sorting.

Installation

Add mldd.min.js to your project and load it.

Initialize LMDD instance with your preferred options, and make sure to listen to the relevant events and respond accordingly

lmdd.init(document.getElementById('yourMarkupID'),{yourOptionsObject});

        Example #1 - Simple Sortable Lists

        Note LMDD is markup oriented, meaning your HTML markup determines the way drag and drop behaviour:

        Elements marked with class="your-draggable-class" will be draggable

        Elements marked with class="your-container-class" will act as containers and you will be able to drop draggables inside them

        HTML markup:

        <div id="simple-sortable-1">
  <div class="container">
      <div class="orange rectangle"></div>
      <div class="light-green rectangle"></div>
      <div class="pink rectangle"></div>
      <div class="deep-purple rectangle"></div>
      <div class="light-blue rectangle"></div>
      <div class="lime rectangle"></div>
  </div>
</div>

        Javascript:

        lmdd.init(document.getElementById('yourMarkupID'),{yourOptionsObject});

        Example #2 - Containers, sortable, and draggable items

        LMDD is markup oriented, meaning you use the markup to tell the app what to act like...

        • html element marked with 'lmdd-container' class will act as a drop target
        • html element marked with 'lmdd-draggable' class will set the element as draggable so you can move it around and drop it in containers
        • html element marked with 'lmdd-sortable' class means this is a sortable item (you can drop the draggable before or after this element
